# Angular Observable Patterns

An Angular project demonstrating RxJS Observable patterns, reactive programming concepts, and data stream management.

## Concepts Covered

- Observable creation and subscription
- Operators (map, filter, switchMap, etc.)
- Subject and BehaviorSubject patterns
- Async pipe usage in templates
- Error handling in streams
- Student data model with reactive patterns

## Tech Stack

- Angular 17+
- TypeScript
- RxJS

## Getting Started

### Prerequisites
- Node.js 18+
- Angular CLI

### Installation
```bash
npm install
ng serve
```

Visit `http://localhost:4200`
